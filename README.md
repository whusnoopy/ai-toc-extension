# AI Chat TOC Pro

一款为 AI 聊天平台提供**精致目录导航**的浏览器扩展，支持 ChatGPT、Google Gemini 和 Claude。

## ✨ 功能特性

- 🎯 **自动提取目录** - 智能识别聊天内容中的标题（H2/H3），自动生成目录
- 🎨 **精致 UI 设计** - 渐变玻璃态背景，深度适配亮色/暗色主题
- 🖱️ **拖拽移动** - 目录面板可自由拖拽，随时调整位置
- ⚡ **快速导航** - 点击目录项快速跳转到对应内容
- 📱 **响应式布局** - 自适应各种屏幕尺寸
- 🔄 **实时更新** - 内容动态更新时自动刷新目录

## 🚀 支持平台

- [ChatGPT](https://chatgpt.com) / [OpenAI Chat](https://chat.openai.com)
- [Google Gemini](https://gemini.google.com)
- [Claude](https://claude.ai)

## 📦 安装指南

### Chrome 浏览器
1. 下载扩展文件到本地
2. 打开 `chrome://extensions/`
3. 启用右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本扩展文件夹
6. 完成！访问支持的 AI 聊天平台即可使用

## 💡 使用说明

1. **自动获取目录** - 在 AI 聊天中发送消息或粘贴内容时，扩展会自动检测标题并生成目录
2. **快速导航** - 点击目录中的任何项目，页面会自动滚动到对应位置
3. **移动面板** - 点击并拖拽目录面板移动到您喜欢的位置
4. **暗色适配** - 自动适配系统或网站的暗色主题

## 🎨 设计特点

- **玻璃态效果** - 现代化的毛玻璃设计风格
- **智能主题** - 自动检测并适配亮色/暗色主题
- **流畅交互** - 平滑的过渡动画和交互反馈
- **清晰排版** - 层次分明的标题等级显示（H2/H3）

## 📝 文件结构

```
ai-toc-extension/
├── manifest.json     # 扩展配置文件

├── style.css         # UI 样式（主题、动画、布局）
└── README.md         # 本文件
```
